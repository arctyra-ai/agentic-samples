## Adding Back Integrations

The open-source version is fully functional without integrations. To add them back:

### CRM (HubSpot, Salesforce, etc.)
1. Create `/app/api/scorecard/register/route.ts`
2. Add your CRM SDK to package.json
3. Call the API in `ScorecardApp.tsx` → `handleEmailSubmit`

### Email (Resend, SendGrid, etc.)
1. Create `/app/api/scorecard/submit/route.ts`
2. Add your email SDK to package.json
3. Call the API in `ScorecardApp.tsx` → `handleComplete`

### Analytics
Replace stub in `lib/analytics.ts` with your preferred analytics provider.
